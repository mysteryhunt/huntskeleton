However, some emoji are not used as letters, but rather as “commands” (as suggested by the flavortext), which change the translation of one or more subsequent letters:

  • 🌌 (Milky Way Emoji): The following letters are Roman numerals, which index into the alphabet. e.g., 🌌VIIII is 8, or H.
  • 👩 (Woman Emoji): The following letter must be decoded with ROT13. e.g., 👩R is E.
  • 🎸 (Guitar Emoji): The following letter must be decoded using the Atbash cipher. e.g., 🎸I is R.
  • 🇱🇾 (Flag for Libya Emoji): The following letter must be replaced with its name of the letter. e.g., 🇱🇾X should be read as EX.
  • 🎮 (Video Game Emoji): The following two letters should be added together, mod-26. e.g., 🎮AB is C, and 🎮YD is B.

Decoding the main recipe works exactly the same way, only it leads to much longer strings. The lengths of the strings are all multiples of 5:

100 = 20 x 5 160 = 32 x 5 85 = 17 x 5 95 = 19 x 5 185 = 37 x 5 165 = 33 x 5 110 = 22 x 5 135 = 27 x 5

These are the same as the number of little squares for each of the eight rectangle shapes in the serving suggestion.
